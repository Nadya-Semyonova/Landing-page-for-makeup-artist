# Landing page for makeup artist

Одностраничный сайт-портфолио для визажиста-гримёра. Проект выполнен на React с использованием TypeScript и Vite. Адаптивный дизайн, плавная навигация, галерея работ и блок с услугами.
https://nadya-semyonova.github.io/Landing-page-for-makeup-artist/

## Основные функции

- **Адаптивный дизайн:** Корректно отображается на мобильных устройствах, планшетах и десктопах.
- **Плавная навигация:** Клик по пунктам меню плавно прокручивает страницу к соответствующей секции.
- **Мобильное меню:** На устройствах с маленьким экраном навигация сворачивается в удобное выдвижное меню.
- **Интерактивная галерея:** Изображения в портфолио увеличиваются при наведении, показывая название работы.
- **Блок с услугами:** Переключаемые категории услуг с отображением соответствующих цен.
- **Компоненты shadcn/ui:** Все базовые элементы интерфейса (кнопки, карточки, вкладки) построены на основе библиотеки shadcn/ui, что обеспечивает единый стиль и высокую кастомизируемость.

## Стек технологий

- **Фреймворк:** React + TypeScript
- **Сборщик:** Vite
- **Стилизация:** Tailwind CSS
- **UI-компоненты:** shadcn/ui
- **Иконки:** Lucide React
